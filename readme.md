# WebryMenu for jQuery
レスポンシブデザインに対応したサイドメニューを簡単に実装できるjQueryのプラグインです。
## 特徴
* モバイル用とデスクトップ用にそれぞれコードを用意しなくても1つのコードで両方のサイドメニューを簡単実装できる！
* BootStrapや、一般的な既存のページにも簡単に適用できると思う。
* サイドメニューだけスクロールできる
* サイドメニューは慣性スクロールに対応(タッチデバイス時)
* サイドメニューを開いたときの横スクロール問題？に対応(タッチデバイス時)
* スライドアニメーションが滑らか！(CSS3アニメーションとjQueryのanimateメソッドを自動切り替え)
* タッチデバイス時のボタンの反応が快適
* サイドメニューを開いている時にメインコンテンツ部分のどこをタップ(クリック)してもメニューが閉じる

## 使い方
[WebryMenu for jQuery 本家サイト](http://webryone.jp/jquery-plugin/webrymenu/)にサンプルや使い方を書いてます。
